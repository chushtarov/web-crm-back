# web-crm-back
Этот проект представляет собой серверную часть для веб-приложения CRM с чатом. Он обеспечивает управление данными и обмен сообщениями между пользователями.

## Краткое описание проекта
web-crm-back - это серверная часть CRM-системы с чатом. Она предоставляет API для взаимодействия с клиентской частью и обеспечивает хранение данных о студентах, курсах и сообщениях в чате.

## Использованный стек:
- Node.js
- Express
- MongoDB
- CORS
- Аутентификация: JSON Web Tokens (JWT)
- WebSocket.io

## Запуск проекта
1. Клонируйте репозиторий: git clone https://github.com/Astamirov/web-crm-back.git
2. Установите зависимости: npm install
3. Запустите сервер: npx nodemon
4. Теперь серверная часть проекта "web-crm-back" готова к использованию и может обрабатывать запросы от клиентской части приложения CRM с чатом.
